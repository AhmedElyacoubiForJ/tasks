# Regressionstests

> Dieser Bereich enthält alle projektspezifischen API‑Regressionstests.
> Die Tests laufen gegen eine laufende Anwendung und prüfen Endpunkte, Fehlerfälle und Use‑Cases automatisiert.

## Struktur

- **_basics/**  
  Grundlagen für API‑Tests, z.B. REST‑Assured‑Einführung.

- **README.md**  
  Einstiegspunkt mit Zielen, Voraussetzungen und Ausführung.

- **regression-tests-overview.md**  
  Warum Regressionstests notwendig sind.

- **regression-tests-dependencies.md**  
  Technische Abhängigkeiten und Setup.

- **regression-tests-execution.md**  
  Wie Tests ausgeführt werden (Einzeltests, Suites, Full Suite).

- **regression-tests-structure.md**  
  Übersicht aller Testklassen und Suites.

- **regression-tests-ci.md**  
  Platzhalter für zukünftige CI‑Automatisierung.

## Zweck

- Automatisierte Prüfung aller API‑Endpunkte
- Klare Trennung zwischen Grundlagen und projektspezifischen Tests
- Erweiterbar für CI‑Pipelines und zusätzliche Testarten
