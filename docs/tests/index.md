# Tests

Dieser Bereich enthält alle Testdokumentationen des Projekts. Er ist in Grundlagen und projektspezifische Regressionstests unterteilt.

## Bereiche

- **_basics/**  
  Grundlagen für API‑Tests, z.B. Einführung in REST‑Assured.

- **regression/**  
  Projektspezifische API‑Regressionstests mit Struktur, Ausführung, Abhängigkeiten und CI‑Vorbereitung.

## Zweck

- Grundlagen vermitteln die Testkonzepte unabhängig vom Projekt.
- Die Regressionstests prüfen die API automatisiert über echte HTTP‑Aufrufe.
- Die Struktur ist erweiterbar für zusätzliche Testarten und CI‑Pipelines.
