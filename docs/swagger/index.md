# Swagger / OpenAPI Übersicht

> Die OpenAPI‑Spezifikation beschreibt die öffentliche API der Anwendung.  
> Sie wird im Ordner `docs/swagger/` abgelegt und kann manuell oder automatisiert erzeugt werden.

## Dateien

- **openapi.yaml** — Haupt‑Spezifikation  
- **openapi.json** — JSON‑Variante  
- **OpenAPI-Exportieren.md** — Anleitung für manuellen Export  
- **OpenApi-Export-CI-Ready.md** — CI‑Roadmap  

## Ziele

- konsistente API‑Dokumentation  
- klare Trennung zwischen Code und API‑Spec  
- Grundlage für Swagger‑UI, Client‑Generatoren und CI‑Pipelines  

## Speicherort der Spezifikation

```
docs/swagger/
```

Die Dateien können versioniert und unabhängig vom Code gepflegt werden.
